# Technical Projects Portfolio

Welcome to my portfolio of data science and machine learning projects. Below is a comprehensive summary of 10 core projects spanning predictive modeling, generative AI applications, and advanced statistical analysis designed for the banking and financial technology sectors.

---

### 1. Enterprise Applied Credit Card Fraud Analysis & Modeling
* **Description:** Developed an end-to-end machine learning solution to detect fraudulent transactions using a massive dataset of 15 million rows. Addressed severe class imbalance using SMOTE and engineered robust transaction behavioral features.
* **Tech Stack:** Python, Jupyter Lab, XGBoost, Scikit-learn, Pandas, SMOTE
* **Links:** [Detailed Code Repository](https://github.com/hemachandra-hd/github.io/tree/main/projects/01-credit-card-fraud-analysis)
